
# Clifford Attractors

R code to generate beautiful high resolution Clifford attractors. The points of a
Clifford attractor are defined by

xₙ₊₁ = sin(a yₙ) + c cos(a xₙ)
yₙ₊₁ = sin(b xₙ) + d cos(b yₙ)


where *a*, *b*, *c*, *d* are constants.
